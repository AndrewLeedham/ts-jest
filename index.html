ts-jest

TSJest (ts-jest) is a TypeScript preprocessor with source map support for Jest that lets you use Jest to test projects written in TypeScript.

It supports all features of TypeScript (even those which Babel 7 typescript plugin does not) and has an internal cache for your tests to run faster.
